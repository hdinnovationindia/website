The Fastest Method (Find and Replace)
1. Open Replace: Press Ctrl + H (Windows) or Cmd + Option + F (Mac).
2. Turn on Regex: Click the .* icon in the search box (or press Alt + R / Cmd + Option + R).
3. Run the Command:
    In the Find box, type: ^\s*[\r\n]
    Leave the Replace box completely empty.
4. Click the Replace All button
